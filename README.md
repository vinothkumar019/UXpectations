## Project Overview  
The Placement Management System (PMS) is a UI/UX design project that digitizes and simplifies the entire campus recruitment process.  
The system is designed for **students, placement officers, and recruiters** to interact on a unified platform.

This project includes:  
- User research  
- Information architecture  
- Wireframes  
- High-fidelity UI  
- Animations & micro-interactions  
- Final interactive prototype  

---

## Problem Statement  
Traditional placement processes have issues like:  
- Manual data handling  
- Miscommunication between students & officers  
- Repetitive form filling  
- No centralized job/internship updates  
- Poor tracking of applied companies  

The PMS system solves this with a **centralized, easy-to-use, modern digital platform**.

## Design Process  
Understand → Research → Analyze → Ideate → Wireframe → Design → Prototype

---

## User Research  

### Target Users  
- Final-year students  
- Placement officers  
- Recruiters  
- Training & internship coordinators  

### Key Insights  
- Students require a clear job listing section  
- Recruiters want a verified student database  
- Placement officers need dashboards to manage registrations  
- Users want simple navigation and clean UI  

---

## Problem Areas Identified  
- No real-time updates for job postings  
- Students miss deadlines  
- Placement officers struggle to manage large student data  
- Recruiters face difficulty verifying eligibility  

---

## Project Goals  
- Fast & transparent communication  
- Smooth job application experience  
- Simple verification system for officers  
- Organized dashboards with analytics
